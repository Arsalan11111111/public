Lims Delays

New fields:

  1. Analysis
    • Date change: Box checked if the dates (date sample and date sample receipt) has change
    • Commercial lead date : this date is calculated by taking the highest duration configured in the parameters pack which has the highest commercial lead time.
    • Commercial warning date : this date is calculated with the duration linked to the parameters pack which has the highest commercial lead time.
    • Is commercial lead date : is true if in the life cycle of this analysis, it was marked as out of time.
    • Is commercial warning date : is true if in the life cycle of this analysis, it was marked as warning time.
  2. Test
    • Date wip : date when the test is set to the status "WIP".
    • Commercial lead date: linked to the commercial lead date from the analysis.
    • Commercial warning date: linked to the commercial warning date from the analysis.
    • Technical lead date: linked to the technical lead date from the results with the smallest technical lead date.
    • Technical warning date: linked to the technical warning date from the results with the smallest technical lead date.
    • Technical lead time: linked to the technical warning time from the results with the smallest technical lead time.
    • Technical warning time: linked to the technical warning time from the results with the smallest technical lead time.
    • Date WIP: This is the date when the test's state change to "WIP"
  3. Results (all types)
    • Technical lead date: This is an expiration date.
    • Technical warning date: This is an alert date.
    • Technical lead time: This is an expiration time.
    • Technical warning time: This is an alert time.
    • Is technical out of time : is true if in the life cycle of this result, it was marked as out of time.
    • Is technical warning date : is true if in the life cycle of this result, it was marked as warning time.

    4. Configuration(s):

  4. Laboratory
    • Default Lead Time: This Lead time used by default if the lead time of the model isn't fill
    • Default Warning Time: This Warning Time used by default if the Warning Time of the model isn't fill
    • Date for compute Warning Time: A selection of date (Date Sample, Date Sample Receipt)
    • Date End for compute Warning time: A selection of date (Date Start Analysis, Date end Analysis, Date report Analysis, Actual Date)
    • Stage Warning Time Filter: A selection of stage (See lims base --> Section Analysis Stage).
  5. Method
    • Due time: this is the due time necessary to use the method
  6. Parameter Characteristic
    • Technical Lead Time: This is an expiration time.
    • Technical Warning time: It's used to set the warning time of a analysis when the warning time of this one is set
  7. Parameter pack
    • Commercial Lead Time: This is an expiration time.
    • Commercial Warning time: It's used to set the Warning time of the analysis

    8. About calculation of date

    Lead date : date start + lead time

    Warning date : date start + lead time - warning date
