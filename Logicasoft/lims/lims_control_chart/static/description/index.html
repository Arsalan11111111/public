LIMS Control Chart

Lims is a management system of laboratory data.

This module need to be use with LIMS Base by Logicasoft

  • Structures :

    • New Model :

      • Lims control chart line

        Object on Method Parameter Characteristic, lines can be define as
        • Target
        • Limit+1
        • Limit-1
        • Limit+2
        • Limit-2
        • Measure : Can be define in object but not use as template by the module (useless)

      • Lims control chart point

        When a result is created, all lines defined in Method Parameter Characteristic will be copied to the result. Method Parameter The characteristic must be marked as "Control card" on its configuration. Note: Only the measure type point will not be copied to the result. It will be created when a numerical result has been entered.

      • this module add new menu

        • Master Data > Control Chart Points
        • Master Data > Control Chart Lines
        • Control Chart New root menu for the lims
        • Control Chart > Control Chart Graph

    • Altered Model :

      • Lims Method Parameter Characteristic, Format : NU

        Boolean : is_control_chart

        control chart lines : several can be define, an created on the from view

        There are a field with the sum of control chart pointOnly the measure 'type'

      • Lims analysis result, Format : NU

        New mechanism : when a result from a Lims Method Parameter Characteristic marked as Control Chart, all the control chart lines define will be copied on the result, except the lines with measure type.

        New mechanism : when an operator insert a result, the result will create a control chart point.

  • Configuration :

    In menu Master Data > Parameter Characteristic

    In tree view : the field 'is_control_chart' Control Chart is displayed Only for the Parameter with format Nu

    In from View : the field 'Toggle' 'is_control_chart' Control Chart is displayed

    When Control Chart is True, a Control Chart notebook is visible

    In this notebook, it is possible to define the different lines for the control card.

  • Usage :

    • In menu Master Data > Parameter Characteristic

      In tree view : the field 'is_control_chart' Control Chart is displayed Only for the Parameter with format Nu

      In from View : the field 'Toggle' 'is_control_chart' Control Chart is displayed

      When Control Chart is True, a Control Chart notebook is visible

      In this notebook, it is possible to define the different lines for the control card.

    • In menu Control Chart > Control Chart Graph

      All Lims Method Parameter Characteristic with Control Chart is displayed

      The control chart graph can be displayed with the button on each 'Lims Method Parameter Characteristic'

    • In menu Master Data > Control Chart Points

      All Control chart point, are displayed

    • In menu Master Data > Control Chart Lines

      All Control chart line, are displayed
