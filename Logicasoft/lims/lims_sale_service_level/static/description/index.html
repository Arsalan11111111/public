Lims Sale Service level

New menu:

  • (Module Sale) Configuration --> Partner Service Type
  • (Module Sale) Configuration --> Service Type

New models:

  • Partner Service Type
    • Partner: The partner
    • Markup: Box checked if the partner service need to be marked
    • Service Type: (See section Service type)
    • Product (Parameter Pack): Product
  • Service Type
    • Sequence : position in list
    • Name: Name of the service
    • Markup: Box checked if the service need to be marked
    • Markup Fees: Product

New fields:

  1. Analysis Request
    • Service Type: (See section Service type)
  2. Partner
    • Service Type: (See section Service type)
  3. Sale order
    • Service Type: (See section Service type)

New features:

It is possible to change the percentage margin on the sold products in the Lims. Either via the general configuration on the analysis request. Or by configuring the margin of a specific product according to a customer and a type of service. Note: the margin of the customer has priority over the margin of the request.
