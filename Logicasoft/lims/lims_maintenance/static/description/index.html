Lims Maintenance

New field:

  1. Equipment traceability type :

    Add a menu in Equipment menu

    • You can add a traceability type to an equipment

  2. Equipment laboratory :

    Add fields in Equipment

    • is_laboratory : boolean which indicates that it is used by the laboratory
    • laboraory_state : selection ('draft', 'in service', 'out of service') mark the equipment if it can currently be used by the laboratory
    • laboratory_id : link with the laboratory on which the equipment depends

  3. Equipment :

    Add a field to link and equipment with :

    • SOP
    • Batch
    • Results

  4. Equipments :

    Add a field to link several equipments with (for configuration) :

    • Method
    • Method parameter characteristic
    • Results

New features:

  1. Manage laboratory equipments
    • You can link devices if they are indicated as "is_laboratory" to :
      • Method: Display a list of equipments available on sop
      • Method parameter characteristic : You can set LOQ an more for result when an equipment is linked to the result
  2. Temporarily declare equipment as out of service for the laboratory :
    • Only If equipment is_laboratory is True and laboratory_state = in_service : you can assign it to results, tests, batches
