Lims Analysis Recurrence

Main use: Allows you to duplicate an analysis already performed and to schedule it at regular intervals.

Objective

Simplification of the work of planning routine analyses. This module adds only one action available on the Lims analyses, this action is available from creation which allows multiplying easily analyses.

Access Menu :

The configuration is available for users with the Lims manager role. The menu can be found in the Lims application on any seen form of an analysis via the action button.

Details of the possibilities of the duplication action:

Available configuration fields:

  1. Interval: Allows you to define the creation interval, if the interval is set to 2 it will generate an element every other period (if day, it will be a generation of every other day).
  2. Up to: Defines the limit of the recurrence, either via the option :
    1. "Number of repetitions" stops when the number of analyses is reached. (configured via the account field)
    2. "End date" stops once the end date is reached when creating recurring analyses. (by entering the date in the Repeat until field that appears when the corresponding mode is selected)
    3. "Period:"
      1. Day (no additional options)
      2. Week
          3. options day of the week (Monday, ..., Sunday)
      3. Month
          4. options (date of the month or day of the month)
        1. date of the month : value between 1 and 31 days
        2. day of the month :
          1. On: (1st, 2nd, 3rd, 4th, last)
          2. day of the week (Monday, ..., Sunday)
      4. Year (no additional options)
