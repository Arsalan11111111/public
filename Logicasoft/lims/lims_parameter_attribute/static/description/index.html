LIMS Parameter Attribute

Lims is a management system of laboratory data.

This module add :

  • 1 Models:

    • 1.1 Configuration model

      • Lims parameter attribute

        • Link a attribute type and attributes values with a result selection value.

      • Lims parameter attribute Type

        • Add a selection on result selection value.

      • Lims parameter attribute type value

        • Add attribute type value, can be added to a result select value.
        • One attribute type value by attribute Type can been added.
        • The result selection value must have an Attribute type configured.

    • 1.2 configuration model alteration (Lims base)

      • Lims parameter

        • add is_attribute boolean, mark the parameter "With attributes" (active the functionality)

      • Lims result value

        • add attribute_ids, link the value with attributes
        • add is_attribute_print boolean, print on report option for this value

      • Lims analysis sel result

        • add attribute_values_ids, link the attributes with the result in sop (Can be acceded with the wizard)
        • add rel_is_attribute boolean, linked with parameter if True, the wizard is available

  • 2 New menu:

    • Master Data

      • Parameter Attribute
      • Parameter Attribute Type
      • Parameter Attribute Type Value

  • 3 New features:

    • Main Feature

      You can now add attribute to the value from a result SE (only)
